[`microsoft/vscode-codicons`](https://github.com/microsoft/vscode-codicons/blob/4ae4e6135009694671cafc84564f2a4e7818d317/src/icons/chat-sparkle.svg).

under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/). The SVG was
modified only by replacing `currentColor` with Mozilla's `context-fill` so it
inherits Zotero's native sidebar icon color.

`codex-logo.svg` is the Mono variant of the Codex (OpenAI) icon from
[`glincker/thesvg`](https://github.com/glincker/thesvg/blob/main/public/icons/codex-openai/mono.svg).
It is used for the selected state and rendered through a CSS mask so it follows
Zotero's current text color. Licensed under the project's MIT license.
